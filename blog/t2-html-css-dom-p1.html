Technical Blog

HTML

HTML is the frame on which websites are built, it is the same as thinking about the frame of a building or the chassis of a car.

DOM

The document object model is a tool used by developers to make changes to your website in real time. It helps devs by filling in information such as tags, that we may have missed.

CSS

CSS is used to decorate or style the building or the car. We can choose the types of materials to adorn our building or car with.

BOX

The box is how we view and present informration on a webpage. All boxes on a weboage have a border, margin, padding and then the content. It helps with positioning and formatting content on a webpage.

Boxify

When we look at a website, we can boxify the design into a template of, well, boxes. This helps us to understand how a website was made. It is important also as it allows us to plan how a website will be built.
