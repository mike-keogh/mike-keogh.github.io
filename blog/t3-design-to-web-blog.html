Design To Web

A responsive site is a site that adjusts its content and size depending on the size of the screen that is viewing it. This is important as it allows the developer to have one set of code that allows them to have a website which is viewable on all screen sizes. In the past developers, would have to make mobile specific sites.

Frameworks are tools that developers can use like skeleton or bootstrap that let the developer apply a grid based system which comes integrated with its own class names and utilities. This means that it can be simpler to create websites from scratch but can mean that sometimes the developer can be restricted in what they can achieve within the framework.
