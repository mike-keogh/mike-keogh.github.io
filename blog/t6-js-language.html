Javascript Language

()

In javascript, the parentheses denote a parameter. In JS we usually find the parentheses after a declared function. In this instance, we can call a function with a different value each time by changing the value in the parentheses when we call the function. This allows us to use the same code for many different tasks. We can also use parentheses to surround a conditional statements. We can also use parentheses to enforce mathematical order of operations eg BEDMAS.

[]

The brackets can mean different things in javascript. The first could be that the values inside of these brackets are an array. The second use could be that the brackets are used to access a property of an object through bracket notation eg object['property'].

{}

Braces can be used to construct an object, or enclose blocks of code.

' ' and " "

The use of single vs double quotes can depend greatly on the situation in which they are used. Some libraries may require a certain type of quote to be used. Whilst the use of both may be necessary in order to make the code readable. The quotes would be used in either case to declare a string or to access a property in an array in bracket notation.
