Scope

In javascript, scope is the way in which the program reads variables. In javascript if a variable is defined outside of a function, it is said to have global scope and can be read and used by any other part of the program. If a variable has local scope, then it means that a variable has been defined inside a function and it can only be used by the function in which it was defined.
